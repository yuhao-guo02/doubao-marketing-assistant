# 美团AI营销助手

基于豆包AI的智能营销内容生成平台，支持多渠道营销内容生成和用户行为数据分析。

## 功能特性

- 🎨 多渠道内容生成（海报大图、Banner、信息流）
- 🤖 AI智能生成文案和图片
- 📊 用户行为数据可视化分析
- 📱 响应式设计

## 部署到Railway

1. Fork此仓库到GitHub
2. 在Railway中创建新项目
3. 连接GitHub仓库
4. 设置环境变量：`DOUBAO_API_KEY`
5. 部署完成

## 本地运行

```bash
pip install -r requirements.txt
python app.py
```

访问 http://localhost:8080
